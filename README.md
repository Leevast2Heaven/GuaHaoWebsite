# GuaHaoWebsite
简单的php+mysql挂号系统后台

# 一、安装说明：

1. 修改config.php中的数据库配置

2. 导入sql文件到数据库中（utf8）

3. 挂号网站：gh.html

4. 登录后台：index.php（为安全考虑，建议更改后台入口文件名）


# 二、系统说明

网上在线挂号预约管理系统使用PHP+MySql开发，适用于任何使用php5和mysql5的空间。

后台账号：admin   后台密码：123456
